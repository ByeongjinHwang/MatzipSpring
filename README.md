# Matzip

# 맛집 등록 사이트 "Matzip"

### Web Outline

- 사용자가 원하는 맛집을 직접 등록하여 공유 할 수 있는 사이트</br>
- 카카오맵을 이용해 음식점 등록 및 메뉴 사진 업로드 가능 </br> 
- 로그인 및 회원가입 제공</br>
- 즐겨찾기 제공</br>
